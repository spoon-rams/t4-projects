The Development Corner

Overview

This documentation page provides an overview of the custom Fordham Rebrand content types developed for use across landing pages and promotional experiences. Each content type is designed to support flexible, visually engaging page layouts while aligning with Fordham’s refreshed brand direction. The documentation outlines the purpose, key features, and potential future enhancements for each component, giving developers and content editors a clear reference for how these content types function, where they can be used, and how they may continue to evolve.

Fordham Rebrand: Hero Banner

Description:

Refreshed the hero banner with a modern gradient overlay to improve visual depth and readability. Removed the dark text container, allowing the headline and supporting copy to sit directly over the imagery for a cleaner, more immersive design. Updated the call-to-action buttons to align with the new brand color palette, ensuring a cohesive and contemporary user experience.

Content Type Features:

  • Five image backgrounds that change randomly on load.
  • Custom title text.
  • Custom description text.
  • Three call-to-action button options.

Future Possible Content Type Features:

  • Custom font sizes and colors.
  • Custom text content width and height.
  • Custom margins and padding.
  • Custom overlay colors.
  • Custom text location

Fordham Rebrand: Two Column Hero

Description:

Developed a modern split-layout hero banner that combines professional imagery with bold, high-impact typography to establish a strong visual hierarchy. The design balances aspirational photography with concise messaging, using a high-contrast color palette to reinforce the brand identity and improve readability. The layout was optimized to create an engaging first impression while effectively directing users' attention to the site's key value proposition.

Content Type Features:

  • Five image backgrounds that change randomly on load.
  • Custom text background colors.
  • Three title text options for text customization with color options.
  • Three call-to-action button options.

Future Possible Content Type Features:

  • Custom font sizes and more color options.
  • More text background color options
  • Custom margins and padding.

Fordham Rebrand: Hero Carousel

Description:

Developed a reusable hero carousel component to showcase multiple promotional messages within a single, engaging interface. The implementation combines responsive full-width imagery, bold headline typography, branded call-to-action buttons, and intuitive carousel controls to create a visually impactful landing experience. The component was designed with reusability, and responsive behavior in mind, making it adaptable for future campaigns and content updates.

Content Type Features:

  • Animations for text enablement.
  • Custom image background.
  • Three title text options for text customization with color options.
  • Three call-to-action button options.

Future Possible Content Type Features:

  • Custom font sizes and more color options.
  • More text background color options
  • Custom margins and padding.
